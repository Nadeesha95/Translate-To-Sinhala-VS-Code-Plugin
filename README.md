        <h2>How to Use</h2>
        <p>Getting feedback on your code is incredibly simple:</p>
        <ol>
            <li>Open any code file (<code>.js</code>, <code>.py</code>, <code>.php</code>, etc.).</li>
            <li>At the top of the editor, click the <strong> Analyze Code with AI (Sinhala)</strong> link.</li>
            <li>The extension will animate a scan down your file and display the results directly inline, complete with gutter icons and translated suggestions.</li>
            <li>Hover over any underlined code to see a detailed explanation in both English and Sinhala.</li>
            <li>When you're done, click the <strong> Clear Analysis</strong> link to hide the decorations.</li>
        </ol>
